Getting Started Commands

/fa -r-start-<RundenID>
Startet eine Runde. Die RundenID erhälst du von einem Leiter.

/fa -r-end
Beendet eine Runde. Weitere Kills & Wins werden nicht gezählt.

/fa -r-restart
Löscht alle Daten von der derzeitigen Runde aus dem Excel Sheet. Kann nicht rückgängig gemacht werden!
